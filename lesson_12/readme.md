### Модифицировать приложение интернет-магазин
1. В информации товара - кнопка “купить”
2. При клике на “купить” ниже появляется форма оформления заказа с следующими полями:
- ФИО покупателя
- Город (выбор из списка)
- Склад Новой почты для отправки
- Наложенный платеж или оплата банковской картой
- Количество приобретаемой продукции
- Комментарий к заказу
3. Реализовать проверку всех пользовательских данных при подтверждении заказа- обязательные поля заполнены. В противном случае, выводить ошибку на страницу
4. Выводить информацию о заказе на страницу (информация о товаре и о доставке)
